Câu 1: Phân biệt display:none, visibility:hidden và opacity:0

  • display: none: Xóa hoàn toàn khỏi layout → không chiếm không gian → gây reflow
  • visibility: hidden: Ẩn nhưng vẫn chiếm không gian → không reflow
  • opacity: 0: Trong suốt nhưng vẫn chiếm không gian + vẫn nhận sự kiện

Câu 2: Trỏ Domain về Hosting

• Nameserver: Thay đổi NS → thời gian cập nhật 4–48 giờ
• A Record: Chỉ sửa IP → cập nhật nhanh hơn (5–60 phút)
→ A Record cập nhật nhanh hơn

Câu 3: Viết lại bằng async/await

async function getData() {
  try {
    const res = await fetch('https://api.example.com/data');
    const data = await res.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}
